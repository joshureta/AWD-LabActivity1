Donation Drive


This website was created by Joshua Ureta, a student from the FEU Institute of Technology. The purpose of this website is to demonstrate the concepts of Programming Fundamentals using JavaScript. The main functionality of this website allows users to input the amount of money they wish to donate. It then records and displays the history of all donations made. This feature not only showcases basic JavaScript concepts but also provides a simple and interactive way to track contributions of a small donation drive.
